# Makefile Workflow Plugin

A focused skill for designing or updating Makefiles with consistent targets and low-noise output.

## Components

| Component           | Type  | Description                                    |
| ------------------- | ----- | ---------------------------------------------- |
| `makefile-workflow` | Skill | Guides creation of workflow-oriented Makefiles |

## Installation

```bash
/plugin install jaeyeom/claude-toolbox/plugins/makefile-workflow
```

## Usage

Ask for Makefile help and the skill will apply:

- "Create a Makefile for this Go repo"
- "Add check targets with quiet output"
- "Update our Makefile to use check-\* strict targets"
